==================
Applications
==================

Applications are layer between user's goals and same goals accomplished. pyLCI applications are typically same as desktop applications, meaning the each of them is mean to perform one function/set of similar functions, and perform it well.
There can be an application for any goal that wants to achieve. 

Applications bundled with the default install are:

   * :ref:`apps_wpa_cli`
   * :ref:`apps_network`
   * :ref:`apps_mocp`
   * :ref:`apps_volume`
   * :ref:`apps_shutdown`
   * :ref:`apps_i2ctools`
